CSS学习(12)-浮动

浮动最初只用于在块级文本内浮动图像，但是现在它已成为在网页上创建多列布局的常用工具之一。

常规流布局

常规流布局是指在不对页面进行任何布局控制时，浏览器默认的 HTML 布局方式。

浮动

浮动可以被用来实现整个网站页面的布局，通过设置浮动使元素横向排列

属性值：

  1. none：表明元素不进行浮动
  2. left：表明元素必须浮动在其所在的块容器左侧
  3. right：表明元素必须浮动在其所在的块容器右侧

浮动特点

  1. 浮动元素会脱离正常的文档布局流，当元素浮动之后，它会被移出正常的文档流，然后向左或者向右平移，一直平移直到碰到了父容器的边框，或者碰到另外一个浮动元素。
  2. 浮动元素只会影响后面的元素，不影响之前的元素
  3. 浮动元素会覆盖正常的文档布局流之上

清除浮动

clear

一个浮动元素被移出正常文档流，其他元素会受到浮动元素的影响，可以通过clear属性来清除浮动影响

属性值：

  1. none：元素不会清除浮动
  2. left：元素会清除左浮动并向下移动
  3. right：元素会清除右浮动并向下移动
  4. both：元素会清除左右浮动并向下移动

如果存在一个盒子同时包含了一个浮动元素和其他元素，可以通过下面三种方式清除浮动

clearfix

向包含浮动内容及其本身的盒子后方插入一些生成的内容，并将生成的内容清除浮动。这与在浮动盒子后手动添加诸如<div>元素，并设置其样式为 clear:both 是等效的。

overflow

将包裹元素的overflow属性设置为除 visible 外的其他值。这样设置相当于创建了BFC，让浮动元素包含在 BFC 及其背景之内。

display：flow-root

将包裹元素的display属性设置为 flow-root。这样设置相当于创建了BFC，定义格式化上下文的根元素。
