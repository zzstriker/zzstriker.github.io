CSS学习(13)-定位

定位允许从正常的文档流布局中取出元素，改变布局中一些盒子的位置。

定位

position 属性用于指定一个元素在文档中的定位方式。top，right，bottom和 left属性则决定了该元素的最终位置。

static

HTML 元素默认定位是static，static不受top，right，bottom和 left属性影响，static只是将元素放入它在文档布局流中的正常位置

relative

相对定位与静态定位非常相似，占据在正常的文档流中，可以通过top，right，bottom和 left属性修改该元素的位置。

absolute

绝对定位的元素脱离了文档流，没有设置top，right，bottom和 left属性的时候，绝对定位会显示在正常文档流的上面，后面的元素会占据他的位置，如果设置top，right，bottom和 left属性，绝对定位元素相对于最近的非static祖先元素定位，如果没有这个祖先元素，将相对于<html>元素定位

fixed

元素会被移出正常文档流，并不为元素预留空间，而是通过指定元素相对于屏幕视口（viewport）的位置来指定元素位置。元素的位置在屏幕滚动时不会改变。可以通过top，right，bottom和 left属性决定该元素的位置。

sticky

粘性定位可以被认为是相对定位和固定定位的混合，须指定top，right，bottom和 left任一属性粘性定位才会生效，否则其行为与相对定位相同。在视口滚动到某个阈值之前，元素为相对定位。之后，元素将固定在与顶部距离的阈值位置。当父元素逐渐消失在视口时，粘性定位的元素也逐渐消失在视口。

z-index

HTML 页面通常被认为是二维的，但前面的浮动和定位会使元素产生重叠。当元素开始重叠，可以通过z-index来决定哪个元素在顶部。
