CSS学习(11)-Display

每个 HTML 元素都有一个默认的 display 值，具体取决于元素类型，大多数默认值为 block 和 inline。

display

display属性用于指定 HTML 元素在网页上的显示方式。

inline

display属性设置为inline的元素不会从新行开始，它将占用剩余可用的屏幕宽度，因此不能设置inline元素的width和height，垂直方向的margin、padding、border也不会生效，水平方向的margin、padding、border会被应用。

block

display属性设置为block的元素会从新行开始，它将占用新行可用的屏幕宽度，，设置block元素的width和height决定占用屏幕的宽度和高度，margin、padding、border会被应用，如果没有指定width，block元素默认占用一整行。

inline-block

display属性设置为inline-block的元素，表现形式和inline元素一样，但是可以设置width和height，也可以设置垂直方向的margin、padding、border

table

将display属性设置为table使元素的行为类似于表格，因此可以不使用 HTML 的table元素和相应的元素创建一个表

flex

display属性设置为flex用于创建横向或纵向的一维页面布局

grid

display属性设置为grid用于同时在两个维度上把元素按行和列排列整齐

inherit

将display属性设置为inherit使元素继承其父元素的display属性，如果在<div>元素中有一个<span>元素，给<span>元素一个inherit属性值，它将从inline变成block元素

none

当display属性设置为none时，元素在页面中仍存在，但是不显示出来，也不占据页面空间

visibility

visibility属性可以显示或隐藏 HTML 元素，作用和display: none相似，但是不会更改页面布局

属性值：

  1. visible：元素可见
  2. hidden：元素不可见，但会影响页面布局

overflow

在盒子无法容纳下太多的内容的时候会发生溢出，overflow属性是控制溢出的一种方式，它告诉浏览器你想怎样处理溢出。

属性值：

  1. visible：内容不被裁剪，有可能会渲染到盒子内边距外面
  2. hidden：内容被裁剪到适应盒子内边距，不提供滚动条，也不支持用户滚动，但是内容可以通过js控制滚动，盒子本质还是一个可以滚动的容器
  3. clip：内容被裁剪到适应盒子内边距，禁止所有滚动，该盒子不是一个可以滚动的容器
  4. scroll：内容被裁剪到适应盒子内边距，无论是否裁剪了内容，浏览器总是显示滚动条
  5. auto：如果内容在盒子内边距内正好显示，它看起来和visible相同，如果内容溢出，则浏览器提供滚动条
