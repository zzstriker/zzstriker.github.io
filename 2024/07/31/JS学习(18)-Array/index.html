JS学习(18)-Array

JavaScript 中数组是一组有序的数据，数组中每个槽位可以存储任意类型的数据。数组也是动态大小的，会随着数据添加而自动增长。JavaScript 中数组的索引是从 0 开始，数组的复制操作是浅拷贝。

Array

JavaScript中的Array对象是一个用于创建和操作数组的全局对象。数组是一种高阶的数据结构，可以存储多个值并按索引进行访问。

数组字面量

构造函数

Array() 构造函数用于创建 Array 对象，调用 Array() 时可以使用或不使用 new都会创建一个新的 Array 实例。

  1. elementN：根据给定的元素创建数组，但是当仅有一个参数且为数字时除外

  2. arrayLength：创建该属性设置的长度的空数组

静态方法

  1. Array.from()：用于将类数组结构转换为数组实例

  2. Array.fromAsync()：与from()方法类似，但是可以操作异步可迭代对象

  3. Array.isArray()：用于确定传递的值是否是一个数组

  4. Array.of()：通过可变数量的参数创建一个新的 Array 实例，而不考虑参数的数量或类型

实例属性

  1. length：表示该数组中元素的个数

迭代器方法

  1. keys()：返回数组索引的迭代器

  2. values()：返回数组元素的迭代器

  3. entries()：返回索引/值对的迭代器
