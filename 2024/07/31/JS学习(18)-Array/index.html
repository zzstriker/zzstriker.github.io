JS学习(18)-Array

JavaScript 中数组是一组有序的数据，数组中每个槽位可以存储任意类型的数据。数组也是动态大小的，会随着数据添加而自动增长。JavaScript 中数组的索引是从 0 开始，数组的复制操作是浅拷贝。

Array

JavaScript中的Array对象是一个用于创建和操作数组的全局对象。数组是一种高阶的数据结构，可以存储多个值并按索引进行访问。

数组字面量

构造函数

Array() 构造函数用于创建 Array 对象，调用 Array() 时可以使用或不使用 new都会创建一个新的 Array 实例。

  1. elementN：根据给定的元素创建数组，但是当仅有一个参数且为数字时除外

  2. arrayLength：创建该属性设置的长度的空数组

静态方法

  1. Array.from()：用于将类数组结构转换为数组实例

    • arrayLike：想要转换成数组的类数组对象或可迭代对象。
    • mapFn（可选）：如果指定了该参数，则新数组中的每个元素会执行该回调函数。
    • thisArg（可选）：执行 mapFn 函数时 this 对象。

  2. Array.fromAsync()：与from()方法类似，但是可以操作异步可迭代对象

  3. Array.isArray()：用于确定传递的值是否是一个数组

  4. Array.of()：通过可变数量的参数创建一个新的 Array 实例，而不考虑参数的数量或类型

实例属性

  1. length：表示该数组中元素的个数

实例方法

迭代器方法

  1. keys()：返回数组索引的迭代器

  2. values()：返回数组元素的迭代器

  3. entries()：返回索引/值对的迭代器

复制填充方法

  1. fill()：向一个已有的数组中插入全部或部分相同的值

    • value：用来填充数组元素的值。
    • start（可选）：开始填充的位置，默认为 0。
    • end（可选）：停止填充的位置（不包括该位置），默认为 this.length。

  2. copyWithin()：复制数组的一部分到同一数组中的另一个位置，并返回它，而不修改其大小。

    • target：开始替换的目标位置。
    • start（可选）：要复制的元素序列的起始位置，默认为 0。
    • end（可选）：要复制的元素序列的结束位置（不包括该位置），默认为数组的长度。

转换方法

  1. toLocaleString()：返回一个字符串表示数组的元素，每个元素通过调用它们自己的 toLocaleString 方法转换为字符串，并且使用特定于语言环境的字符串

    • locales（可选）：一种 BCP 47 语言标签的字符串或字符串数组，表示所需的语言环境。
    • options（可选）：一个对象，其属性是配置语言环境中某些特定格式化行为的属性。

  2. toString()：返回一个字符串，表示指定的数组及其元素。

  3. valueOf()：用于返回数组对象本身

  4. join()：将一个数组的所有元素连接成一个字符串并返回这个字符串，用逗号或指定的分隔符字符串分隔。如果数组只有一个元素，那么将返回该元素而不使用分隔符。

    • separator（可选）：指定一个字符串来分隔数组的每个元素。如果省略，数组元素用逗号（,）分隔。

栈方法

  1. pop()：删除数组末尾的一个元素，返回该元素，会更改数组的长度

  2. push()：将指定的元素添加到数组的末尾，并返回新的数组长度。

队列方法

  1. shift()：从数组中删除第一个元素，并返回该元素的值。会更改数组的长度

  2. unshift()：将指定元素添加到数组的开头，并返回数组的新长度

迭代方法

  1. forEach()：对数组的每个元素执行一次给定的函数，没有返回值。

    • callback：对每个元素执行的函数。它接受三个参数
      • currentValue：数组中正在处理的当前元素
      • index（可选）：数组中正在处理的当前元素的索引
      • array（可选）：forEach 方法正在操作的数组
    • thisArg（可选）：执行回调时用作 this 的值

  2. every()：测试数组中的所有元素是否都通过了指定函数的测试。该方法返回一个布尔值

    • callback：对每个元素执行的函数。它接受三个参数
      • currentValue：数组中正在处理的当前元素
      • index（可选）：数组中正在处理的当前元素的索引
      • array（可选）：every 方法正在操作的数组
    • thisArg（可选）：执行回调时用作 this 的值

  3. filter()：创建一个新数组，新数组中的元素是通过测试的所有数组元素。

    • callback：对每个元素执行的函数。它接受三个参数
      • currentValue：数组中正在处理的当前元素
      • index（可选）：数组中正在处理的当前元素的索引
      • array（可选）：filter 方法正在操作的数组
    • thisArg（可选）：执行回调时用作 this 的值

  4. map()：创建一个新数组，新数组中的元素是通过调用提供的函数对每个元素进行处理的结果。

    • callback：对每个元素执行的函数。它接受三个参数
      • currentValue：数组中正在处理的当前元素
      • index（可选）：数组中正在处理的当前元素的索引
      • array（可选）：map 方法正在操作的数组
    • thisArg（可选）：执行回调时用作 this 的值

  5. some()：测试数组中是否至少有一个元素通过了由提供的函数实现的测试。如果在数组中找到一个元素使得提供的函数返回 true，则返回 true；否则返回 false。它不会修改数组。

    • callback：对每个元素执行的函数。它接受三个参数
      • currentValue：数组中正在处理的当前元素
      • index（可选）：数组中正在处理的当前元素的索引
      • array（可选）：some 方法正在操作的数组
    • thisArg（可选）：执行回调时用作 this 的值

归并方法

  1. reduce()：将数组中的所有元素按某种规则累计成一个单一的值。这个方法会遍历数组中的每个元素，执行提供的函数，并将结果汇总为最终的单一值。reduce 方法不会改变原数组。

    • callback：对每个元素执行的函数。它接受四个参数
      • accumulator：上一次调用 callback 时返回的值。第一次调用时，如果指定了 initialValue 则为指定的值，否则为 array[0] 的值。
      • currentValue：当前元素的值。在第一次调用时，如果指定了 initialValue，则为 array[0] 的值，否则为 array[1]。
      • currentIndex（可选）：数组中正在处理的当前元素的索引
      • array（可选）：reduce 方法正在操作的数组
    • initialValue（可选）：指定第一次调用 callback 时 accumulator 的初始值。如果没有提供，accumulator 将使用数组中的array[0] 的值，currentValue 是array[1] 的值。

  2. reduceRight()：与 reduce() 方法类似，都是用于将数组中的所有元素按某种规则累计成一个单一的值。不同的是，reduceRight() 从数组的最后一个元素开始向前遍历，直到第一个元素。

排序方法

  1. reverse()：反转数组中的元素，并返回数组

  2. sort()：对数组的元素进行排序并返回数组，默认排序是将元素转换为字符串，然后按照它们的 UTF-16 码元值升序排序。

    • compareFn（可选）：一个用于排序的比较函数。它接受两个参数 a 和 b，并根据它们的顺序来确定排序顺序。如果未提供 compareFn，则默认按照字符串的 Unicode 编码顺序进行排序。

  3. toReversed()：返回一个新数组，反转数组中的元素，而原始数组不会被修改。

  4. toSorted()：返回一个新数组，反转数组中的元素，而原始数组不会被修改。

    • compareFn（可选）：一个用于排序的比较函数。它接受两个参数 a 和 b，并根据它们的顺序来排序元素。如果未提供 compareFn，则默认为升序排序。

操作方法

  1. concat()：用于合并两个或更多的数组或值，并返回一个新的数组。该方法不会改变原始数组，而是返回一个新的数组。

    • value1, value2, ..., valueN：要合并到数组中的值或数组。可以是任意数量的数组或值

  2. slice()：从数组中返回选定的元素，生成一个新数组。它不会修改原始数组，而是返回一个包含选定元素的新数组。

    • begin（可选）：从哪个索引开始选择元素。包括该索引。默认为 0。
    • end（可选）：在哪个索引停止选择元素。排除该索引。默认为数组的长度。

  3. splice()：用于在数组中添加、删除或替换元素，并返回被删除的元素。splice() 方法会修改原始数组。

    • start：从哪个索引开始改变数组。负值表示从数组的末尾开始计算位置。
    • deleteCount（可选）：要删除的元素个数。如果省略或为 0，则不会删除元素。
    • item1, item2, ..., itemN（可选）：要添加到数组中的元素。如果没有提供，则只删除元素。

  4. toSpliced()：返回一个新数组，该数组是通过对原始数组进行指定的删除和插入操作得到的，而原始数组不会被修改。

    • start：从哪个索引开始改变数组。负值表示从数组的末尾开始计算位置。
    • deleteCount（可选）：要删除的元素个数。如果省略或为 0，则不会删除元素。
    • item1, item2, ..., itemN（可选）：要添加到数组中的元素。如果没有提供，则只删除元素。

搜索和位置方法

  1. indexOf()：返回数组中第一次出现给定元素的下标，如果不存在则返回 -1

    • searchElement：要查找的元素。
    • fromIndex（可选）：开始搜索的索引位置。默认为 0。如果 fromIndex 为负数，它会从数组的末尾开始计算位置。

  2. lastIndexOf()：返回数组中给定元素最后一次出现的索引，如果不存在则返回 -1

    • searchElement：要查找的元素。
    • fromIndex（可选）：开始搜索的索引位置。默认为 0。如果 fromIndex 为负数，它会从数组的末尾开始计算位置。

  3. includes()：判断数组是否包含指定的元素。如果数组中包含该元素，则返回 true；否则返回 false。

    • searchElement：要查找的元素。
    • fromIndex（可选）：开始搜索的索引位置。默认为 0。如果 fromIndex 为负数，它会从数组的末尾开始计算位置。

  4. find()：返回数组中第一个满足提供测试函数的元素。如果没有找到满足条件的元素，则返回 undefined。

    • callback：对每个元素执行的函数。它接受三个参数
      • currentValue：数组中正在处理的当前元素
      • index（可选）：数组中正在处理的当前元素的索引
      • array（可选）：find 方法正在操作的数组
    • thisArg（可选）：执行回调时用作 this 的值

  5. findIndex()：返回数组中第一个满足提供测试函数的元素的索引。如果没有找到符合条件的元素，则返回 -1。

    • callback：对每个元素执行的函数。它接受三个参数
      • currentValue：数组中正在处理的当前元素
      • index（可选）：数组中正在处理的当前元素的索引
      • array（可选）：findIndex 方法正在操作的数组
    • thisArg（可选）：执行回调时用作 this 的值

  6. findLast()：返回数组中最后一个满足提供测试函数的元素。如果没有找到满足条件的元素，则返回 undefined。

    • callback：对每个元素执行的函数。它接受三个参数
      • currentValue：数组中正在处理的当前元素
      • index（可选）：数组中正在处理的当前元素的索引
      • array（可选）：findLast 方法正在操作的数组
    • thisArg（可选）：执行回调时用作 this 的值

  7. findLastIndex()：返回数组中最后一个满足提供测试函数的元素的索引。如果没有找到满足条件的元素，则返回 -1。

    • callback：对每个元素执行的函数。它接受三个参数
      • currentValue：数组中正在处理的当前元素
      • index（可选）：数组中正在处理的当前元素的索引
      • array（可选）：findLastIndex 方法正在操作的数组
    • thisArg（可选）：执行回调时用作 this 的值

扁平化方法

  1. flat()：创建一个新的数组，并根据指定深度递归地将所有子数组元素拼接到新的数组中。

    • depth（可选）：一个整数，表示递归的深度。默认值是 1。如果 depth 为 Infinity，则会递归所有嵌套的数组。

  2. flatMap()：对数组中的每个元素应用给定的回调函数，然后将结果展开一级，返回一个新数组。

    • callback：一个生产新数组元素的函数。接收三个参数：
      • currentValue: 当前数组元素。
      • index（可选）：当前元素的索引。
      • array（可选）：原始数组。
    • thisArg（可选）：执行 callback 函数时 this 的值

其他操作方法

  1. at()：使用正负整数索引来访问数组中的元素

    • index: 一个整数，表示要访问的元素的索引。可以是正数或负数。

  2. with()：返回一个新的数组，该数组是对原始数组进行指定索引位置的元素替换后的结果，而不修改原始数组。

    • index: 要替换的元素的索引。如果是负数，表示从数组末尾开始计数。
    • value: 要替换的新值。
