JS学习(9)-Number

在 JavaScript 中，Number 类型是用于表示数字的基本数据类型。它可以表示整数、浮点数、正数、负数、以及一些特殊的值如 Infinity 和 NaN。JavaScript 中所有数字都是以64位双精度浮点数（IEEE 754标准）存储的。

Number

Number 构造函数包含常量和处理数值的方法。其他类型的值可以使用 Number() 函数转换为数字。

数字字面量

JavaScript 支持多种表示数字的方式：

  1. 十进制：常见的数字表示方法。

  2. 浮点数：包含小数点的数字。

  3. 科学计数法：用于表示非常大的或非常小的数字。

  4. 八进制：以 0o 或 0O 开头。

  5. 十六进制：以 0x 或 0X 开头。

  6. 二进制：以 0b 或 0B 开头。

特殊值

  1. Infinity 和 -Infinity：表示无穷大和负无穷大。

  2. NaN(Not-a-Number)：表示一个非数字的值，通常是非法数学操作的结果。

Number对象

JavaScript 提供了一个Number对象，用于包装数字类型，并提供了一些静态属性和方法。

构造函数

Number 是对应数值的引用类型。要创建一个 Number 对象，就使用 Number 构造函数并传入一个 数值

  1. value：可选，任意类型，表示数字的任意值，默认为0

静态属性

  1. Number.EPSILON：判断两个浮点数是否“足够接近”，从而认为它们相等，因为在 JavaScript 中，直接比较两个浮点数是否相等可能会导致意外结果，因为浮点数的表示精度有限。

  2. Number.MAX_SAFE_VALUE：JavaScript 中最大的安全整数，即 $ 2^{53} - 1 $ 。

  3. Number.MAX_VALUE：JavaScript 中能表示的最大数。

  4. Number.MIN_SAFE_VALUE：JavaScript 中最小的安全整数，即 $ - 2^{53} + 1 $ 。

  5. Number.MIN_VALUE：JavaScript 中能表示的最小的正数。

  6. Number.NaN：表示 NaN。

  7. Number.NEGATIVE_INFINITY：表示负无穷大。

  8. Number.POSITIVE_INFINITY：表示正无穷大。

静态方法

  1. Number.isFinite(value)：检查值是否是有限的（非 Infinity、-Infinity 或 NaN）。

  2. Number.isInteger(value)：检查值是否是整数。

  3. Number.isNaN(value)：检查值是否是 NaN。

  4. Number.isSafeInteger(value)：判断传入的值是否为安全整数 ( $ -2^{53} + 1 $ 至 $ 2^{53} - 1 $ 之间)。

  5. Number.parseFloat(string)：解析一个参数（必要时先转换为字符串）为浮点数。

  6. Number.parseInt(string, [radix])：将字符串解析为指定基数的十进制整数，radix 是进制基数。

实例方法

  1. toExponential(fractionDigits)：将数字格式化为指数计数法的字符串。

  2. toFixed(digits)：将数字格式化为固定小数位的字符串。

  3. toLocaleString(digits)：将数字格式化为固定小数位的字符串。

  4. toPrecision(precision)：将数字格式化为本地化字符串表示

  5. toString([radix])：将数字转换为字符串，radix 是进制基数。

  6. valueOf()：返回指定对象的原始值，通常是隐式调用。
