JS学习(15)-Symbol

Symbol 是 ES6 引入的一种新的原始数据类型。每个 Symbol 值都是唯一的，用于生成独一无二的标识符。它是通过调用 Symbol 函数创建的，Symbol 函数不能作为构造函数使用。

Symbol

每个从 Symbol() 返回的 symbol 值都是唯一的。一个 symbol 值能作为对象属性的标识符

构造函数

Symbol() 构造函数返回一个 symbol 类型的值，但是它并不完全支持构造函数的语法，因为它不支持 new Symbol() 语法。

静态方法

  1. Symbol.for()：Symbol.for() 方法会根据给定的键，来从运行时的 Symbol 注册表中找到对应的 Symbol，如果找到了，则返回它，否则，新建一个与该键关联的 Symbol，并放入全局 Symbol 注册表中。

    Symbol.for 与 Symbol 这两种写法，都会生成新的 Symbol。它们的区别是：前者会被登记在全局环境中供搜索，后者不会。

  2. Symbol.keyFor()：Symbol.keyFor(sym) 方法用来获取全局 symbol 注册表中与某个 symbol 关联的键。

静态属性

  1. Symbol.asyncIterator：指定了一个对象的默认异步迭代器。如果一个对象设置了这个属性，它就是异步可迭代对象，可用于for await…of 循环。

  2. Symbol.hasInstance：用于判断某对象是否为某构造器的实例。

  3. Symbol.isConcatSpreadable：用于配置对象作为 Array.prototype.concat() 方法的参数时是否展开其数组元素。

  4. Sumbol.iterator：为每一个对象定义了默认的迭代器。该迭代器可以被 for…of 循环使用。

实例方法

  1. toString：返回当前 symbol 对象的字符串表示。

  2. valueOf()：返回当前 symbol 对象的值。

实例属性

  1. description：是一个只读属性，它会返回 Symbol 对象的可选描述的字符串。
