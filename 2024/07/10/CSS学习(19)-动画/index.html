CSS学习(19)-动画

在CSS中，transform、transition 和 animation 是用于实现视觉效果和交互动画的关键属性。

转换

CSS 转换可以在不影响正常文档流的情况下改变作用内容的位置。通常使用 transform 和 transform-origin 属性来定义 CSS 转换。

  1. transform-origin：指定原点的位置，默认是元素的中心
  2. transform：指定作用在元素上的转换，多个转换使用空格分隔，转换从右到左执行

transform-style

transform-style 属性设置元素是位于 3D 空间中还是平面中。

属性值：

  1. flat：设置元素位于平面中
  2. preserve-3d：设置元素位于 3D 空间中

translate

translate()方法将元素从当前位置进行平移

属性值 说明
translateX(x) 指定对象 X 轴（水平方向）的平移
translateY(y) 指定对象 Y 轴（垂直方向）的平移
translateZ(z) 指定对象 Z 轴的平移
translate(x,y) 指定对象的 2D 平移。两个参数分别对应 X 轴和 Y 轴。如果第二个参数未提供，则默认值为 0
translate3d(x,y,z) 指定对象的 3D 位移。三个参数分别对应 X 轴、 Y 轴和 Z 轴，参数不允许省略

rotate

rotate() 方法根据给定的度数顺时针或逆时针旋转元素，元素围绕 transform-origin 属性定义的点旋转。

属性值 说明
rotateX(angle) 指定对象在 X 轴上的旋转角度
rotateY(angle) 指定对象在 Y 轴上的旋转角度
rotateZ(angle) 指定对象在 Z 轴上的旋转角度
rotate(angle) 指定对象的 2D 旋转，需先有 transform-origin 属性的定义
rotate3d(x,y,z,angle) 指定对象的 3D 旋转角度，其中前 3 个参数分别表示旋转的方向 X、Y、Z，第 4 个参数表示旋转的角度，参数不允许省略

scale

scale()方法根据给定的参数按比例扩缩元素的大小。

属性值 说明
scaleX(x) 指定对象 X 轴的（水平方向）缩放
scaleY(y) 指定对象 Y 轴的（垂直方向）缩放
scaleZ(z) 指定对象的 Z 轴缩放
scale(x,y) 指定对象的 2D 缩放。两个分别对应 X 轴和 Y 轴。如果第二个参数未提供，则默认取第一个参数的值
scale3d(x,y,z) 指定对象的 3D 缩放。三个参数对应 X 轴、 Y 轴和 Z 轴，参数不允许省略

skew

skew()方法按给定的角度沿 X 轴和 Y 轴倾斜元素。

属性值 说明
skewX(angle) 指定对象 X 轴的（水平方向）倾斜
skewY(angle) 指定对象 Y 轴的（水平方向）倾斜
skew(x-angle,y-angle) 指定对象 X、Y 轴的倾斜，两个参数分别对应 X 轴和 Y 轴，如果第二个参数未提供，则默认值为 0

matrix

matrix(a, b, c, d, e, f)方法通过矩阵进行复杂的2D变换。

参数解释：

  1. a：水平缩放（scaleX）
  2. b：垂直倾斜（skewY）
  3. c：水平倾斜（skewX）
  4. d：垂直缩放（scaleY）
  5. e：水平平移（translateX）
  6. f：垂直平移（translateY）

过渡

CSS 过渡通常使用简写属性 transition 控制。transition 属性可以被指定为一个或多个 CSS 属性的过渡效果，多个属性之间用逗号进行分隔。如果没有指定CSS 属性和过渡效果时长，那么过渡动画是没有效果的。

transition 简写属性 CSS 语法如下：

transition-property

transition-property 用于设定过渡的属性名称，规定设置过渡效果的 CSS 属性的名称。

属性值：

  1. none：没有过渡动画。
  2. all：所有可被动画的属性都表现出过渡动画。
  3. 属性名称：CSS 属性的名称。

transition-duration

transition-duration 用于设定过渡动画所需的时间，规定完成过渡效果需要多少秒或毫秒，默认为 0s。

transition-delay

transition-delay 用于设定过渡效果开始作用之前需要等待的时间，取值为正时会延迟一段时间来响应过渡效果；取值为负时会导致过渡立即开始。默认为 0s。

transition-timing-function

transition-timing-function 用于设定过渡的速度曲线，规定速度效果的速度曲线，默认为 ease。

属性值 说明
linear 线性效果，动画从头到尾的速率是相同的
ease 表示缓动函数 cubic-bezier(0.25, 0.1, 0.25, 1)，动画以低速开始，然后加快，结束前变慢
ease-in 表示缓动函数 cubic-bezier(0.42, 0, 1, 1)，动画以低速开始，然后逐渐加快
ease-out 表示缓动函数 cubic-bezier(0, 0, 0.58, 1)，动画以快速开始，然后逐渐减慢
ease-in-out 表示缓动函数 cubic-bezier(0.42, 0, 0.58, 1)，动画开始类似于ease-in，结束类似于ease-out
step-start 马上转跳到动画结束状态
step-end 保持动画开始状态，直到动画执行时间结束，马上转跳到动画结束状态
steps() 函数接受正值，在过渡时间内步进式过渡
cubic-bezier(a,b,c,d) 贝塞尔曲线，接受四个值来做曲线

动画

CSS 动画通常使用简写属性 animation 控制。animation 属性包含描述动画的样式规则和用于指定动画开始、结束以及中间点样式的关键帧。

animation 简写属性 CSS 语法如下：

animation-name

animation-name 关键帧名称，用于定义 @keyframes 动画的名称，与 @keyframes 定义的动画名称一致。

属性值：

  1. none：无动画效果
  2. <keyframename>：需要绑定到选择器的 keyframes 的名称

animation-duration

animation-duration 动画持续时间，规定元素动画播放完成一个周期所需要的时间，以秒或毫秒计量。默认为 0 表示没有动画效果。

animation-timing-function

animation-timing-function 速度曲线，规定速度效果的速度曲线，默认为 ease。

属性值 说明
linear 线性效果，动画从头到尾的速率是相同的
ease 表示缓动函数 cubic-bezier(0.25, 0.1, 0.25, 1)，动画以低速开始，然后加快，结束前变慢
ease-in 表示缓动函数 cubic-bezier(0.42, 0, 1, 1)，动画以低速开始，然后逐渐加快
ease-out 表示缓动函数 cubic-bezier(0, 0, 0.58, 1)，动画以快速开始，然后逐渐减慢
ease-in-out 表示缓动函数 cubic-bezier(0.42, 0, 0.58, 1)，动画开始类似于ease-in，结束类似于ease-out
step-start 马上转跳到动画结束状态
step-end 保持动画开始状态，直到动画执行时间结束，马上转跳到动画结束状态
steps() 函数接受正值，在过渡时间内步进式过渡
cubic-bezier(a,b,c,d) 贝塞尔曲线，接受四个值来做曲线

animation-delay

animation-delay 属性规定执行动画前的等待时间，以秒或毫秒计量。正值表示动画应在指定的时间量过去后开始，默认值为 0s，表示动画应立即开始，负值会导致动画立即开始，但是从动画循环的某个时间点开始。

animation-iteration-count

animation-iteration-count 属性规定动画的播放次数，可选具体次数或者无限（infinite），默认为 1。

animation-direction

animation-direction 属性规定执行动画前的等待时间，以秒或毫秒计量。正值表示动画应在指定的时间量过去后开始，默认值为 0s，表示动画应立即开始，负值会导致动画立即开始，但是从动画循环的某个时间点开始。

属性值：

  1. normal：动画在每个循环中正向播放。
  2. reverse：动画在每个循环中反向播放。
  3. alternate：动画在每个循环中正反交替播放，第一次迭代是正向播放。
  4. alternate-reverse：动画在每个循环中正反交替播放，第一次迭代是反向播放。

animation-fill-mode

属性值：

  1. none：默认值。动画在执行之前或之后不会对元素应用任何样式
  2. forwards：元素将保留最后一个关键帧设置的样式值（取决于动画方向和动画迭代次数）
  3. backwards：元素将获取第一个关键帧设置的样式值（取决于动画方向），并在动画延迟期间保留该值
  4. both：动画将遵循向前和向后的规则，在两个方向上扩展动画属性

animation-play-state

animation-play-state 属性规定动画的播放状态，用此来控制动画的暂停和继续。

属性值：

  1. running：默认值，动画运行
  2. paused：动画暂停

@keyframes

通过在动画序列中定义关键帧 @keyframes 的样式来控制 CSS 动画序列中的中间步骤。

属性值：

  1. 关键帧名称：名称需符合CSS 标识符命名，关键帧名称需要与动画名称一致
  2. from：等价于 0%
  3. to：等价于 100%
  4. 百分数：动画序列中触发关键帧的时间点，使用百分数来表示
