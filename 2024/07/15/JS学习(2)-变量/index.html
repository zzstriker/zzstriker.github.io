JS学习(2)-变量

变量，就是一个用于存放数值的容器。这个数值可能是一个用于累加计算的数字，或者是一个句子中的字符串。变量的独特之处在于它存放的数值是可以改变的。

变量

变量可以用于保存任何类型的数据。每个变量只不过是一个用于保存任意值的命名占位符。有 3 个关键字可以声明变量：var、const 和let。var在 ECMAScript 的所有版本中都可以使用，而const和let只能在 ECMAScript 6 及更晚的版本中使用。

var 声明

声明变量

要定义变量，可以使用var关键字，后跟变量名（即标识符）。

变量赋值

用 var声明的变量，在变量名之后跟上一个=，然后是变量值，就完成了赋值，如果没有赋值，则变量值为 undefined

变量赋值后，也可以更改变量值

var可以多次声明同一个变量

如果一个变量没有声明，直接赋值，默认是使用了var声明

声明作用域

var 声明的变量具有函数作用域或全局作用域

使用var在一个函数内部定义一个变量，就意味着该变量将在函数退出时被销毁，该变量是局部变量

在函数内没有使用var声明的变量或在函数体外使用var声明的变量，该变量是全局变量

使用var在全局作用域中声明的变量会成为 window 对象的属性

声明提升

使用var声明的变量会自动提升到作用域顶部

let 声明

声明变量

要定义变量，可以使用let关键字，后跟变量名（即标识符）。

变量赋值

用 let声明的变量，在变量名之后跟上一个=，然后是变量值，就完成了赋值，如果没有赋值，则变量值为 undefined

变量赋值后，也可以更改变量值

let不可以多次声明同一个变量

声明作用域

let 声明的变量是块级作用域

let 在全局作用域中声明的变量不会成为 window 对象的属性

暂时性死区

let 声明的变量不会在作用域中提升，变量没有声明之前的区域，被称为暂时性死区

const 声明

const 创建的是常量。常量标识符的命名规则和变量相同。

  1. 常量不可以通过重新赋值改变其值，也不可以在代码运行时重新声明。它必须被初始化为某个值。

  2. 常量的作用域规则与 let 块级作用域变量相同。若省略 const 关键字，则该标识符将被视为变量。

  3. 在同一作用域中，不能使用与变量名或函数名相同的名字来命名常量

字面量

字面量是指在代码中直接写出来的固定值。这些值可以是数字、字符串、布尔值等，直接在代码中使用，而不是通过变量或表达式计算得到的。
