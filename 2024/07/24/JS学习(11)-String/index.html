JS学习(11)-String

在 JavaScript 中，String 类型用于表示和操作文本数据。字符串可以使用单引号 (')、双引号 (")、或者反引号（`）包围。

String

字符串字面量

字符串可以通过字符串字面量创建为原始值

构造函数

String() 构造函数创建 String 对象。当作为函数调用时，它返回 String 类型的原始值。

  1. thing：任何要转换为字符串的内容。

当 String 作为构造函数（使用 new）被调用时，它会创建一个 String 对象，该对象不是原始类型。当 String 作为函数被调用时，它会将参数强制转换为一个字符串原始类型。

实例属性

  1. length：表示字符串中字符的数量

静态方法

  1. String.fromCharCode()：返回由指定的 UTF-16 代码单元序列创建的字符串。

  2. String.fromCodePoint()：返回使用指定的代码点序列创建的字符串。

  3. String.raw()：

实例方法

获取字符串的值

  1. toString()：返回该字符串的值

  2. valueOf()：返回 String 对象的字符串值

获取字符串指定位置的值

  1. charAt()：返回指定位置的字符

  2. charCodeAt()：返回指定索引位置字符的 UTF-16 值

字符串包含方法

  1. idnexOf()：返回第一次出现指定字符串的位置，如果没有找到则返回 -1。

  2. lastIndexOf()：返回指定字符串在字符串中最后一次出现的位置，如果没有找到则返回 -1。

  3. startsWith()：判断当前字符串是否以指定字符串开头，如果是则返回 true，否则返回 false。

  4. endsWith()：判断一个字符串是否以指定字符串结尾，如果是则返回 true，否则返回 false。

  5. includes()：判断一个字符串是否包含另一个字符串，返回布尔值。

字符串操作方法

  1. concat()：连接一个或多个字符串，并返回新的字符串

  2. slice()：提取字符串的一部分，并返回一个新的字符串。不修改原始字符串

  3. subString()：提取字符串中两个指定的索引之间的字符。如果未提供结束索引，则返回到字符串末尾的部分。

重复字符串

  1. repeat()：返回一个新字符串，表示将原字符串重复n次

补充字符串长度

  1. padStart()：用于头部补全。该方法有两个参数，其中第一个参数是一个数字，表示字符串补齐之后的长度；第二个参数是用来补全的字符串。

  2. padEnd()：用于尾部补全。该方法也是接收两个参数，第一个参数是字符串补全生效的最大长度，第二个参数是用来补全的字符串

移出字符串空白字符

  1. trim()：字符串的两端移除空白字符，并返回一个新的字符串，不会修改原始字符串

  2. trimEnd()：字符串的结尾移除空白字符，并返回一个新的字符串，不会修改原始字符串

  3. trimStart()：字符串的开头移除空白字符，并返回一个新的字符串，不会修改原始字符串

字符串大小写转换

  1. toLowerCase()：将字符串转换为小写形式

  2. toLocaleLowerCase()：根据特定区域设置的大小写映射规则，将字符串转换为小写形式并返回

  3. toUpperCase()：将字符串转换为大写形式

  4. toLocaleUpperCase()：根据特定区域设置的大小写映射规则，将字符串转换为大写形式并返回

字符串模式匹配

  1. split()：用于把一个字符串分割成字符串数组。该方法不会改变原始字符串。

    该方法有两个参数：

    • separator：必需。字符串或正则表达式，从该参数指定的地方分割 string。
    • limit：可选。该参数可指定返回的数组的最大长度

  2. search()：在 String 对象中执行正则表达式的搜索，寻找匹配项。

  3. replace()：用于使用指定字符串替换当前字符串中匹配指定正则表达式模式的子字符串，并返回完成替换后的字符串。

  4. replaceAll()：用于使用指定字符串替换所有当前字符串中匹配指定正则表达式模式的子字符串，并返回完成替换后的字符串。

  5. match()：使用指定的正则表达式模式在当前字符串中进行匹配查找，并返回数组形式的查找结果。

  6. matchAll()：返回一个包含所有匹配正则表达式的结果及分组捕获组的迭代器。
