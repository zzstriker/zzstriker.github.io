CSS学习(15)-Flex

Flex是Flexible Box的缩写，是一种用于按行或按列布局元素的一维布局方法。元素可以膨胀以填充额外的空间，收缩以适应更小的空间。

Flex

Flexible Box 模型，通常被称为 flexbox，弹性布局的目的是提供一种更加高效的方式来对容器中的条目进行布局、对齐和分配空间。创建 flex 容器需要把容器的 display 属性设置为 flex。

  1. 主轴（main axis）：是沿着 flex 元素放置的方向延伸的轴，该轴的开始和结束被称为 main start 和 main end。
  2. 交叉轴（cross axis）：是垂直于 flex 元素放置方向的轴。该轴的开始和结束被称为 cross start 和 cross end。
  3. 设置了 display: flex 的父元素，被称之为 flex 容器（flex container）。
  4. 在 flex 容器中表现为弹性的盒子的元素被称之为 flex 项（flex item）。

主轴方向

在 flex 容器中添加 flex-direction 属性可以让我们更改 flex 元素的排列方向。

属性值：

  1. row：表示flex 项从左向右排列。此时水平方向轴为主轴。
  2. row-reverse：表示flex 项从右向左排列。
  3. column：表示flex 项从上向下排列。此时垂直方向轴为主轴。
  4. column-reverse：表示flex 项从下向上排列。

换行

在 flex 容器中添加 flex-wrap 属性用于指定弹性布局中flex 项是否换行。

属性值：

  1. nowrap：表示不换行，所有flex 项单行排列，flex 项可能会溢出。
  2. wrap：表示换行，所有flex 项目多行排列，溢出的flex 项会被放到下一行，按从上向下顺序排列。
  3. wrap-reverse：所有flex 项多行排列，按从下向上顺序排列。

flex-flow

可以将两个属性 flex-direction 和 flex-wrap 组合为简写属性 flex-flow。第一个指定的值为 flex-direction ，第二个指定的值为 flex-wrap。

对齐方式

主轴方向对齐

justify-content 属性定义了flex 项在 主轴上的对齐方式。

属性值：

  1. flex-start：默认值，表示弹性容器flex 项按主轴起点线对齐
  2. flex-end：表示弹性容器flex 项按主轴终点线对齐
  3. center：居中排列
  4. space-between：弹性容器flex 项均匀分布，第一项紧贴主轴起点线，最后一项紧贴主轴终点线，flex 项之间的间隔都相等。要注意特殊情况，当剩余空间为负数或者只有一个项时，效果等同于 flex-start。
  5. space-around：弹性容器flex 项均匀分布，每个项目两侧的间隔相等，相邻项目之间的距离是两个项目之间留白的和。所以，项目之间的间隔比项目与边框的间隔大一倍。要注意特殊情况，当剩余空间为负数或者只有一个项时，效果等同于 center。
  6. space-evenly：弹性容器flex 项均匀分布，所有项目之间及项目与边框之间的距离相等。

交叉轴方向对齐

align-items 属性定义了flex 项在 交叉轴上的对齐方式。

属性值：

  1. stretch：默认值，当flex 项未设置高度或者高度为 atuo 时，会使所有flex 项沿着交叉轴的方向拉伸以填充父容器。如果父容器在交叉轴方向上没有固定高度，则所有flex 项的高度将与最高的flex 项一样。
  2. flex-start：表示弹性容器flex 项按交叉轴起点线对齐。
  3. flex-end：表示弹性容器flex 项按交叉轴终点线对齐。
  4. center：表示与交叉轴的中线对齐。

多交叉轴方向对齐

align-content 属性定义了flex 项在 多根交叉轴上的对齐方式。如果只有一根交叉轴，该属性不起作用。针对盒子内部的元素超过了盒子的宽度出现了换行，flex 项是多行。

属性值：

  1. stretch：默认值，当flex 项未设置高度或者高度为 atuo 时，会使所有flex 项沿着交叉轴的方向拉伸以填充父容器。如果父容器在交叉轴方向上没有固定高度，则所有flex 项的高度将与最高的flex 项一样。
  2. flex-start：表示弹性容器flex 项按交叉轴起点线对齐。
  3. flex-end：表示弹性容器flex 项按交叉轴终点线对齐。
  4. center：表示与交叉轴的中线对齐。
  5. space-between：弹性容器flex 项均匀分布，第一项紧贴主轴起点线，最后一项紧贴主轴终点线，flex 项之间的间隔都相等。要注意特殊情况，当剩余空间为负数或者只有一个项时，效果等同于 flex-start。
  6. space-around：弹性容器flex 项均匀分布，每个项目两侧的间隔相等，相邻项目之间的距离是两个项目之间留白的和。所以，项目之间的间隔比项目与边框的间隔大一倍。要注意特殊情况，当剩余空间为负数或者只有一个项时，效果等同于 center。
  7. space-evenly：弹性容器flex 项均匀分布，所有项目之间及项目与边框之间的距离相等。

flex 项排序

弹性盒子可以通过order改变 flex 项的布局位置的功能，而不会影响到源顺序（即 dom 树里元素的顺序）。

  1. 所有的 flex 项默认的 order 值是0
  2. order 值大的 flex 项比 order 值小的在显示顺序中更靠后。
  3. 拥有相同 order 值的元素按照它们在源代码中出现的顺序进行布局。

flex 项动态尺寸

flex-grow

flex-grow 属性定义flex 项的扩展比例，表示放大的比例，取值必须是正整数，负值无效，默认为 0。Flex 容器会根据flex 项设置的扩展比例作为比率来分配剩余空间，如果所有flex 项的 flex-grow 属性都为 1，则它们将等分剩余空间。如果一个flex 项的 flex-grow 属性为 2，其他为 1，则前者占据的剩余空间将比其他项多一倍。

计算方式：

  1. 假设剩余空间 x（弹性容器宽度与所有flex 项宽度总和之差）
  2. 假设有三个flex 项元素，flex-grow 设定值分别为 a、b 和 c
  3. 每个flex 项元素可以扩展的空间为：a/(a+b+c) * x、b/(a+b+c) * x 和 c/(a+b+c) * x

flex-shrink

flex-shrink 属性定义flex 项的缩小比例，表示缩小的比例，取值必须是正整数，负值无效，默认为 1。如果flex 项宽度之和超过Flex 容器宽度，Flex 容器会根据flex 项设置的缩小比例作为比率来压缩flex 项宽度，以保证不溢出。

计算方式：

  1. 假设三个flex 项的 width 为：w1、w2、w3
  2. 假设三个flex 项的 flex-shrink 为：a、b、c
  3. 计算总压缩权重：sum = a * w1 + b * w2 + c * w3
  4. 计算每个元素压缩率：s1 = a * w1 / sum、s2 = b * w2 / sum、s3 = c * w3 / sum
  5. 计算每个元素宽度：width - 压缩率 * 溢出空间

flex-basis

flex-basis 属性指定flex 项在主轴方向上的初始长度。

注意：

设置 flex-grow 进行分配剩余空间，或者使用 flex-shrink 进行收缩都是在 flex-basis 的基础上进行的

flex

flex 属性是 flex-grow、flex-shrink 和 flex-basis 的简写，默认值为 0 1 auto。后两个属性可选。

flex 项对齐

align-self 属性用于指定flex 项的对齐方式，可覆盖 align-items 属性。

属性值：

  1. auto：默认值，表示继承父元素的 align-items 属性，如果没有父元素，则等同于 stretch。
  2. flex-start：表示弹性容器flex 项按交叉轴起点线对齐。
  3. flex-end：表示弹性容器flex 项按交叉轴终点线对齐。
  4. center：表示与交叉轴的中线对齐。
  5. stretch：当flex 项未设置高度或者高度为 atuo 时，会使所有flex 项沿着交叉轴的方向拉伸以填充父容器。如果父容器在交叉轴方向上没有固定高度，则所有flex 项的高度将与最高的flex 项一样。
