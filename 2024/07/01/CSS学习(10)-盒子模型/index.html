CSS学习(10)-盒子模型

在 CSS 中，所有的元素都被一个个的“盒子”包围着，理解这些“盒子”的基本原理，是我们使用 CSS 实现准确布局、处理元素排列的关键。

盒模型

块盒子和内联盒子

在 CSS 中，我们有几种类型的盒子，一般分为块盒子（block boxes）和内联盒子（inline boxes）。类型指的是盒子在页面流中的行为方式以及与页面上其他盒子的关系。

盒子显示类型

盒子有内部显示（inner display type）和外部显示（outer display type）两种类型。

外部显示类型

一个块盒子在外部显示类型中会表现出以下行为：

  1. 盒子会产生换行
  2. width和height属性可以发挥作用
  3. margin、padding、border会将其他元素从当前盒子周围”推开“
  4. 如果没有指定width，盒子将填充容器中的空间

一个内联盒子在外部显示类型中会表现出以下行为：

  1. 盒子不会产生换行
  2. width和height属性将不起作用
  3. 垂直方向的margin、padding、border会被应用但是不会把其他内联盒子推开。
  4. 水平方向的margin、padding、border会被应用且会把其他内联盒子推开。

内部显示类型

决定了盒子内元素的布局方式。

CSS盒模型

CSS 盒模型整体上适用于块盒子，它定义了盒子的不同部分（外边距、边框、内边距和内容）如何协同工作，以创建一个在页面上可以看到的盒子。内联盒子使用的只是盒模型中定义的部分行为。

CSS 中组成一个块盒子需要：

  1. 内容盒子：显示内容的区域
  2. 内边距盒子：填充位于内容周围的空白处
  3. 边框盒子：边框盒子包住内容和任何填充
  4. 外边距盒子：外边距是最外层，其包裹内容、内边距和边框，作为该盒子与其他元素之间的空白

标准盒模型

在标准模型中，如果给盒子设置width和height，实际设置的是Content box。padding 和 border 再加上设置的宽高一起决定整个盒子的大小。

IE盒模型

在IE盒模型中，所有宽度都是可见宽度，所以内容宽度是该宽度减去border 和 padding。

注意：默认情况下，浏览器使用标准盒模型。

box-sizing

  1. content-box：默认值，标准盒子模型。width和height只包括内容的宽和高，不包括border、padding、margin
  2. border-box：IE盒子模型。width和height包括内容的宽高和 border、padding，不包括margin

外边距

外边距是盒子周围的空间，它会把其他元素从盒子旁边推开，无论使用标准模型还是替代模型，外边距总是在计算可见部分后额外添加。

我们可以使用margin属性控制元素的所有边距，margin 属性是 margin-top、margin-right、margin-bottom 和 margin-left 的简写形式。

margin 简写形式以特定顺序应用属性：上、右、下、左，具体应用方式如下：

  1. 一个值：所有面都会应用。(margin: 20px)
  2. 两个值：第一个值将应用于 top 和 bottom，第二个值将应用于 left 和 right。(margin: 20px 40px)
  3. 三个值：第一个值为 top，第二个值为 left 和 right，第三个值为 bottom。(margin: 20px 40px 30px)
  4. 四个值：第一个值为 top，第二个值为 right，第三个值为 bottom，第四个值为 left。(margin: 20px 40px 30px 40px)

负外边距

外边距属性值可以为正也可以为负。在盒子一侧设置负值会导致盒子和页面上的其他内容重叠。

外边距折叠

根据外边距相接触的两个元素是正边距还是负边距，结果会有所不同：

  1. 两个正外边距将合并为一个外边距。其大小等于最大的单个外边距
  2. 两个负外边距会折叠，并使用最小（离零最远）的值
  3. 如果其中一个外边距为负值，其值将从总值中减去

注意：外边距何时折叠，何时不折叠，由许多规则决定。需要记住的主要一点是，外边距折叠是指在使用外边距创建空间时，如果没有获得预期的空间，就会发生外边距折叠。

边框

元素外边距内就是边框，元素的边框就是围绕元素内容和内边距的一条或多条线。

可以使用border属性一次设置所有四个边框的宽度、颜色和样式。border 属性是 border-top、border-right、border-bottom 和 border-left 的简写形式。

边框样式

如需显示边框，就必须定义border-style

  1. none：默认值，不显示边框
  2. hidden：与none关键字类似，隐藏边框，如果存在其他的重叠边框，边框不会显示
  3. dotted：显示为一系列圆点，圆点半径是border-width值的一半
  4. dashed：显示为一系列短的方形虚线
  5. solid：显示为一条实线
  6. double：显示为一条双实线
  7. groove：显示为有雕刻效果的边框，样式与ridge相反
  8. ridge：显示为有浮雕效果的边框，样式与groove相反
  9. inset：显示为有陷入效果的边框，样式与outset相反
  10. outset：显示为有突出效果的边框，样式与inset相反

边框宽度

边框的宽度是线条的粗细，由 border-width 控制。

  1. thin：细边线
  2. medium：默认值，中等边线
  3. thick：宽边线
  4. 长度值设置边框宽度（px、em、rem、%）

边框颜色

可以使用 border-color 设置框的各侧或每一侧的颜色。默认情况下，它会使用框的当前文本颜色

边框圆角

使用border-radius属性设置边框的圆角，在简写形式中指定每个角的半径，遵循顺序：左上角、右上角、右下角、左下角。

内边距

内边距位于边框和内容区域之间，用于将内容推离边框。与外边距不同，内边距不能为负数。任何应用于元素的背景都会显示在内边距后面。

padding 属性是padding-top、padding-right、padding-bottom 和 padding-left 的简写形式。

阴影

块阴影

box-shadow 属性用于向 HTML 元素的框添加阴影。它适用于块元素和内联元素。

box-shadow 的值顺序如下：

  1. inset（可选）：如果没有指定inset，默认阴影在边框外，即阴影向外扩散。
  2. 水平偏移：正数表示阴影位于元素右边，负数表示阴影位于元素左边。
  3. 垂直偏移：正数表示阴影位于元素下方，负数表示阴影位于元素上方。
  4. 模糊半径（可选）：值越大，模糊面积越大，阴影就越大越淡。不能为负值。默认为 0
  5. 扩散半径（可选）：取正值时，阴影扩大；取负值时，阴影收缩。默认为 0，此时阴影与元素同样大。
  6. 颜色（可选）：任何有效的颜色值。 如果未定义，将使用计算出的文本颜色。

文字阴影

text-shadow 属性用于为文字添加阴影。

text-shadow 的值顺序如下：

  1. 水平偏移：正数表示阴影位于元素右边，负数表示阴影位于元素左边。
  2. 垂直偏移：正数表示阴影位于元素下方，负数表示阴影位于元素上方。
  3. 模糊半径（可选）：值越大，模糊面积越大，阴影就越大越淡。不能为负值。默认为 0
  4. 颜色（可选）：任何有效的颜色值。 如果未定义，将使用计算出的文本颜色。

曲线阴影

如果需要实现沿着图片任何可能曲线的阴影，可以使用drop-shadow过滤器

drop-shadow 的值顺序如下：

  1. 水平偏移：正数表示阴影位于元素右边，负数表示阴影位于元素左边。
  2. 垂直偏移：正数表示阴影位于元素下方，负数表示阴影位于元素上方。
  3. 模糊半径（可选）：值越大，模糊面积越大，阴影就越大越淡。不能为负值。默认为 0
  4. 扩散半径（可选）：取正值时，阴影扩大；取负值时，阴影收缩。默认为 0，此时阴影与元素同样大。
  5. 颜色（可选）：任何有效的颜色值。 如果未定义，将使用计算出的文本颜色。
