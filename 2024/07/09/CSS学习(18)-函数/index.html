CSS学习(18)-函数

CSS 中提供了大量内置函数，每个 CSS 函数都有特定的用途。

函数

变量

CSS 变量可以具有全局或局部范围，变量名必须以两个破折号（–）开头，并且区分大小写。

var()

var()函数用于插入 CSS 变量的值。

全局声明

全局变量可以通过整个文档使用，创建具有全局作用域的变量，需要在:root 选择器内声明它。:root选择器与文档的根元素匹配。

局部声明

局部变量只能在声明它的选择器内使用，创建局部变量，在使用它的选择器内声明它。

颜色函数

rgb()

rgb() 函数将三个取值范围0-255的数字作为参数，表示颜色的红色、绿色和蓝色，并返回RGB 值。

hsl()

hsl() 函数有三个参数，色调、饱和度和亮度值

数学表达式

calc()

calc() 函数允许使用加法、减法、乘法和除法来指定 CSS 属性值。通常用于组合计算具有不同单位的两个 CSS 值。

min()

min() 函数将一组逗号分隔的值作为参数，并返回这些值中的最小值。

max()

max() 函数将一组逗号分隔的值作为参数，并返回这些值中的最大值。

clamp()

clamp() 函数组合了min() 和max() 函数，clamp() 函数有三个参数：最小值，限制值和最大值。

  1. 如果要限制的值小于传递的最小值，则函数将返回最小值。
  2. 如果要限制的值大于传递的最大值，则函数将返回最大值。
  3. 如果要限制的值处于最小值与最大值之间，则函数将返回要限制的原始值。
