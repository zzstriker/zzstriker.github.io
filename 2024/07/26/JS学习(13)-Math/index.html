JS学习(13)-Math

在 JavaScript 中，Math 对象拥有一些数学常数属性和数学函数方法。Math 对象用于 Number 类型，不支持 BigInt。

Math

Math 对象的方法和属性是静态的，可以直接调用而无需创建 Math 对象的实例。

静态属性

  1. Math.PI：圆周率 π 的值（约等于 3.14159）。
  2. Math.E：自然对数的底数 e（约等于 2.718）。
  3. Math.LN2：2 的自然对数（约等于 0.693）。
  4. Math.LN10：10 的自然对数（约等于 2.302）。
  5. Math.LOG2E：以 2 为底的 e 的对数（约等于 1.442）。
  6. Math.LOG10E：以 10 为底的 e 的对数（约等于 0.434）。
  7. Math.SQRT1_2：1/2 的平方根（约等于 0.707）。
  8. Math.SQRT2：2 的平方根（约等于 1.414）。

静态方法

  1. Marh.abs(x)：返回数值的绝对值。

  2. Marh.ceil(x)：向上取整，返回大于等于 x 的最小整数。

  3. Marh.floor(x)：向下取整，返回小于等于 x 的最大整数。

  4. Marh.round(x)：四舍五入，返回最接近 x 的整数。

  5. Marh.random()：返回一个大于等于 0 且小于 1 的伪随机浮点数

  6. Marh.max(...values)：返回给定一组数中的最大值，如果没有参数，则返回 -Infinity。

  7. Marh.min(...values)：返回给定一组数中的最小值，如果没有参数，则返回 Infinity。

  8. Marh.pow(base, exponent)：返回 base 的 exponent 次幂。

  9. Marh.sqrt(x)：返回 x 的平方根。

  10. Marh.cbrt(x)：返回 x 的立方根。

  11. Marh.hypot(...values)：返回所有参数的平方和的平方根

  12. Marh.trunc(x)：返回数字的整数部分，去除小数部分。

  13. Marh.sin(x)：返回 x 的正弦值，x 是以弧度表示的角度。

  14. Marh.cos(x)：返回 x 的余弦值，x 是以弧度表示的角度。

  15. Marh.tan(x)：返回 x 的正切值，x 是以弧度表示的角度。

  16. Marh.asin(x)：返回 x 的反正弦值，返回值在 -π/2 到 π/2 之间。

  17. Marh.acos(x)：返回 x 的反余弦值，返回值在 0 到 π 之间。

  18. Marh.atan(x)：返回 x 的反正切值，返回值在 -π/2 到 π/2 之间。
