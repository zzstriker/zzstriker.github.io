JS学习(30)-BOM

虽然 ECMAScript 把浏览器对象模型（BOM，Browser Object Model）描述为 JavaScript 的核心，但实际上 BOM 是使用 JavaScript 开发 Web 应用程序的核心。BOM 提供了与网页无关的浏览器功能对象。

BOM

BOM 是指浏览器提供的所有对象和接口的集合，用于与浏览器窗口进行交互。通过 BOM，可以操作浏览器窗口、控制浏览器的显示和导航，甚至与用户进行交互。BOM 主要是与浏览器窗口本身及其属性、事件、行为相关的。

window对象

BOM 的核心是 window 对象，表示浏览器的实例。window 对象在浏览器中有两重身份，一个是 ECMAScript 中的 Global 对象，另一个就是浏览器窗口的 JavaScript 接口。

Global

因为 window 对象被复用为 ECMAScript 的 Global 对象，所以通过 var 声明的所有全局变量和函 数都会变成 window 对象的属性和方法。

窗口关系

  1. window.top：指向最顶层窗口的引用，即最高级的窗口对象。
  2. window.parent：指向当前窗口或iframe的父窗口的引用。
  3. window.self：对当前 window 对象的引用。通常直接使用 window 对象即可，但 window.self 可以用来明确表示当前窗口。
  4. window.frames：一个类数组对象，包含了当前窗口中所有子 iframe 的引用。
  5. window.opener：指向打开当前窗口的那个窗口的引用
  6. window.closed：是一个布尔属性，指示当前窗口是否已关闭。
  7. window.name：是一个字符串属性，表示当前窗口的名称。
  8. window.length：表示当前窗口中包含的 iframe 数量
  9. window.open：用于打开一个新的浏览器窗口或标签页。它返回新窗口的 window 对象。
  10. window.close：用于关闭当前窗口，或关闭由脚本打开的窗口。

窗口大小

  1. window.innerWidth：返回窗口内容区域（视口）的宽度
  2. window.innerHeight：返回窗口内容区域（视口）的高度
  3. window.outerWidth：返回浏览器窗口的整体宽度
  4. window.outerHeight：返回浏览器窗口的整体高度

窗口位置

  1. window.scrollX：返回窗口的水平滚动偏移量
  2. window.scrollY：返回垂直滚动偏移量
  3. window.scrollTo(x, y)：将窗口滚动到指定的坐标 (x, y)
  4. window.scrollBy(dx, dy)：根据当前的滚动位置进行滚动

定时器

  1. setTimeout()：接收两个参数：要执行的代码和在执行回调函数前等待的时间（毫秒）。
  2. clearTimeout()：在指定时间到达之前调用clearTimeout()，就可以取消超时任务。在任务执行后再调用clearTimeout()没有效果。
  3. setInterval()：接收两个参数：要执行的代码和下一次执行定时代码的任务添加到队列要等待的时间（毫秒）。
  4. clearInterval()：取消定时

系统对话框

  1. alert()：显示一个警告框，包含一段文本和一个确认按钮。
  2. confirm()：显示一个确认对话框，包含一段文本和“确定”和“取消”按钮。返回 true 或 false。
  3. prompt()：显示一个输入对话框，用户可以在对话框中输入文本。返回用户输入的文本，如果用户取消输入，则返回 null。

location对象

location 提供了当前窗口中加载文档的信息，以及通常的导航功能。它既是 window 的属性，也是 document 的属性，window.location 和 document.location 指向同一个对象。

属性 值 说明
location.hash “#contents” URL 散列值（井号后跟零或多个字符），如果没有则为空字符串
location.host “www.wrox.com:80” 服务器名及端口号
location.hostname “www.wrox.com” 服务器名
location.href “http://www.wrox.com:80/WileyCDA/ ?q=javascript#contents” 当前加载页面的完整 URL
location.pathname “/WileyCDA/” URL 中的路径和文件名
location.port “80” 请求的端口。如果 URL中没有端口，则返回空字符串
location.protocol “http:” 页面使用的协议。通常是"http:“或"https:”
location.search “?q=javascript” URL 的查询字符串。这个字符串以问号开头
location.username “foouser” 域名前指定的用户名
location.password “barpassword” 域名前指定的密码
location.origin “http://www.wrox.com” URL 的源地址。只读

navigator对象

navigator 对象用来表示浏览器状态和身份的对象，它提供了有关浏览器的信息。

  1. navigator.userAgent：返回当前浏览器的用户代理字符串
  2. navigator.language：返回一个字符串，表示用户的首选语言
  3. navigator.onLine：返回布尔值，表示浏览器是否连接到网络
  4. navigator.geolocation：允许网页访问用户的地理位置信息
  5. navigator.cookieEnabled：返回布尔值，表示是否启用了 cookie
  6. navigator.hardwareConcurrency：返回用户计算机上可用于运行线程的逻辑处理器数量
  7. navigator.mediaDevices：返回一个对象，提供与用户媒体输入设备
  8. navigator.clipboard：提供对系统剪贴板的访问功能
  9. navigator.permissions：允许查询和请求用户的权限状态
  10. navigator.storage：访问存储管理 API 的功能，用于检查存储的持久性和使用情况

history对象

history 对象表示当前窗口首次使用以来用户的导航历史记录。history 是 window 的属性， 所以每个 window 都有自己的 history 对象。history 不会暴露用户访问过的 URL， 但可以通过它在不知道实际 URL 的情况下前进和后退。

  1. history.go()：在用户历史记录中沿任何方向导航，可以前进也可以后退。

  2. history.back()：用于模仿浏览器的后退按钮

  3. history.forward()：用于模仿浏览器的前进按钮

  4. history.pushState()：向当前浏览记录栈中添加一条新的历史记录，添加后页面不会重新加载。

    • state：用于存储该 URL 对应的状态对象。该对象可通过 history.state 或 popstate 事件回调中的 event 对象获取。如果不需要这个对象，此处可以填 null。
    • title：新页面的标题
    • url：URL 地址，不允许跨域。

  5. history.replaceState()：向当前浏览记录栈中将当前历史记录替换掉。

    • state：用于存储该 URL 对应的状态对象。该对象可通过 history.state 或 popstate 事件回调中的 event 对象获取。如果不需要这个对象，此处可以填 null。
    • title：新页面的标题
    • url：URL 地址，不允许跨域。

screen对象

screen 对象表示一个屏幕窗口，往往指的是当前正在被渲染的 window 对象，可以使用 window.screen 获取它。

  1. screen.width 和 screen.height：screen.width 返回屏幕的宽度，screen.height 返回屏幕的高度，单位是像素。这两个属性表示整个屏幕的分辨率。
  2. screen.availWidth 和 screen.availHeight：screen.availWidth 返回屏幕可用的宽度，screen.availHeight 返回屏幕可用的高度。这两个属性减去了操作系统中那些占用屏幕空间的元素（如任务栏或 Dock），表示实际可以用来显示内容的区域。
  3. screen.colorDepth：colorDepth 返回显示屏的颜色深度，以位为单位。颜色深度表示显示设备能够支持的每个像素的颜色位数。
  4. screen.pixelDepth：pixelDepth 返回屏幕的像素深度，通常与 colorDepth 相同。这是每个像素可以显示的位数。
  5. screen.orientation：orientation 返回一个 ScreenOrientation 对象，表示屏幕的当前方向（横向或纵向）以及锁定状态。它包含属性和方法来检查和更改屏幕方向。
