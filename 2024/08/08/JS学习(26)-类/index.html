JS学习(26)-类

在 JavaScript 中，class 是一种用于创建对象的语法糖。虽然 JavaScript 的本质是一种基于原型的语言，但 class 语法提供了类似于传统面向对象编程语言的结构，使得定义和使用类更直观、更易读。

类

使用 class 关键字来定义一个类。一个类可以包含构造函数、方法（包括静态方法）和访问器（getter 和 setter）。
