JS学习(21)-Set

Set 是一种集合类型，可以存储任何类型的值，无论是原始值还是对象引用。

Set

Set 对象是值的合集。Set 中的元素只会出现一次，即集合中的元素是唯一的，可以按照插入顺序迭代集合中的元素。

构造函数

  1. iterable（可选）：如果传入一个可迭代对象，它的所有元素将不重复地被添加到新的 Set 中。如果不指定此参数或其值为 null，则新的 Set 为空。

实例属性

  1. size：返回 Map 中元素的数量

实例方法

  1. add()：向 Set 中添加一个新元素。

  2. has()：返回一个布尔值，检查 Set 中是否存在某个值

  3. delete()：从 Set 中删除一个元素，如果该值在集合中

  4. clear()：移除该 Set 中的所有元素。

  5. entries()：返回一个新的迭代器对象，该对象包含了此 Set 中的每个元素的 [value, value] 数组，按插入顺序排列。

  6. forEach()：按插入顺序对该 Set 中的每个值执行一次提供的函数

    • callbackFn：为 Set 中每个元素执行的函数。使用以下参数调用该函数
      • value：每个迭代的值
      • key：每个迭代的键，与value相同
      • set：正在迭代的 Set
    • thisArg：执行 callbackFn 时用作 this 的值

  7. keys()：是values()方法的别名。

  8. values()：返回一个新的迭代器对象，该对象包含 Set 对象中每个元素的值。

  9. union()：并集，返回一个新集合，该集合是两个集合中所有的唯一项元素

  10. intersection()：交集，返回一个新集合，该集合是两个集合都存在的元素

  11. difference()：差集，返回一个新集合，返回集合A中不在集合B中的元素

  12. symmetricDifference()：返回一个新集合，返回只在集合A或集合B中的元素

  13. isSubsetOf()：检查一个集合的所有元素是否都在另一个集合中

  14. isSupersetOf()：检查一个集合是否包含另一个集合中的所有元素

  15. isDisjointFrom()：检查两个集合是否没有任何共同的元素

WeakSet

WeakSet 是 JavaScript 中一种集合类型。WeakSet是 Set 的“兄弟”类型，其 API 也是 Map 的子集。

特性

  1. WeakSet 的元素只能是对象或 Symbol，不能是原始类型的值

  2. WeakSet 中对象的引用为弱引用。如果没有其他的对 WeakSet 中对象的引用存在，那么这些对象会被垃圾回收

  3. 因为 WeakSet 中的值有可能被垃圾回收，所以WeakSet不可迭代

构造函数

  1. iterable（可选）：如果传入了一个可迭代对象，这个对象的所有元素都会被添加到新的 WeakSet 对象中。null 会被视为 undefined。

实例方法

  1. add()：向 WeakSet 中添加一个新对象

  2. has()：检查 WeakSet 中是否存在某个对象

  3. delete()：从 WeakSet 中删除一个对象
