JS学习(31)-DOM

DOM（Document Object Model）是浏览器解析 HTML 或 XML 文档时生成的一种编程接口，通过它，开发者可以访问和操作文档的结构、内容和样式。DOM 将文档表示为一个结构化的树形节点，允许使用脚本语言（如 JavaScript）动态地更改文档内容。

DOM

任何 HTML 或 XML 文档都可以用 DOM 表示为一个由节点构成的层级结构。document 节点表示每个文档的根节点，，根节点的唯一子节点是元素，我们称之为文档元素（documentElement）。文档元素是文档最外层的元素，所有其他元素都存在于这个元素之内。每个文档只能有一个文档元素。

Node 类型

Node 是 DOM 中的基本类型，代表文档树中的所有节点类型。DOM 树是由各种类型的节点组成的层次结构，每个节点都继承自 Node 对象。Node 类型定义了一些通用的属性和方法，这些属性和方法可以被文档中的不同节点类型共享。

属性

  1. nodeType：返回节点的类型，通常用来确定节点的类型。

    • Node.ELEMENT_NODE

    • Node.ATTRIBUTE_NODE

    • Node.TEXT_NODE

    • Node.CDATA_SECTION_NODE

    • Node.ENTITY_REFERENCE_NODE

    • Node.ENTITY_NODE

    • Node.PROCESSING_INSTRUCTION_NODE

    • Node.COMMENT_NODE

    • Node.DOCUMENT_NODE

    • Node.DOCUMENT_TYPE_NODE

    • Node.DOCUMENT_FRAGMENT_NODE

    • Node.NOTATION_NODE

  2. nodeName：返回节点的名称，对于元素节点，它是标签名；对于属性节点，它是属性名；对于文本节点，它是 #text。

  3. nodeValue：返回或设置节点的值，适用于文本节点、注释节点和属性节点。

  4. parentNode：返回当前节点的父节点

  5. childNodes：返回包含当前节点所有子节点的 NodeList，包括元素、文本和注释节点

  6. firstChild和lastChild：返回第一个和最后一个子节点

方法

  1. appendChild(node)：将指定的节点添加为当前节点的最后一个子节点
  2. insertBefore(newNode, referenceNode)：在指定的参考节点之前插入新的子节点
  3. removeChild(node)：从当前节点的子节点列表中移除指定的子节点
  4. replaceChild(newNode, oldNode)：使用新的节点替换现有的子节点
  5. cloneNode(deep)：接收一个布尔值参数，表示是否深复制。在传入 true 参数时，会进行深复制， 即复制节点及其整个子 DOM 树。如果传入 false，则只会复制调用该方法的节点。

Document 类型

Document 类型是 JavaScript 中表示文档节点的类型。在浏览器中，文档对象 document 是 HTMLDocument 的实例（HTMLDocument 继承 Document），表示整个 HTML 页面。document 是 window 对象的属性，因此是一个全局对象。

文档子节点

  1. document.documentElement：访问文档的根元素，即 <html> 标签。
  2. document.head：访问文档的 <head> 元素。
  3. document.body：访问文档的 <body> 元素。

文档信息

  1. document.title：获取或设置文档的标题（位于 <title> 标签中）。
  2. document.URL：当前页面的完整 URL
  3. document.domain：获取或设置页面的域名。
  4. document.referrer：链接到当前页面的那个页面的 URL

定位元素

  1. document.getElementById()：通过元素的 id 属性获取特定元素
  2. document.getElementsByTagName()：获取指定标签名的所有元素，返回一个 HTMLCollection 对象
  3. document.getElementsByName()：根据元素的 name 属性值查找文档中的所有匹配元素，返回一个 HTMLCollection 对象

文档写入

  1. document.write()：接收一个字符串参数，将内容写入文档流
  2. document.writeln()：接收一个字符串参数，将内容写入文档流，还会在字符串末尾追加一个换行符 （\n）。
  3. document.open()：开始对文档的写操作
  4. document.close()：结束对文档的写操作

Element 类型

Element 类型是 DOM 中表示 HTML 或 XML 文档中元素节点的基本类型。它继承自 Node 类型，是所有具体元素节点的基类，每个 HTML 标签在 DOM 中都对应一个 Element 对象。

HTML 元素

所有 HTML 元素都通过 HTMLElement 类型表示，包括其直接实例和间接实例。另外，HTMLElement 直接继承 Element 并增加了一些属性。

  1. id：元素在文档中的唯一标识符
  2. title：包含元素的额外信息
  3. lang：元素内容的语言代码
  4. dir：语言的书写方向（"ltr"表示从左到右，"rtl"表示从右到左）
  5. className：相当于 class 属性，用于指定元素的 CSS 类
