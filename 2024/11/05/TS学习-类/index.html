TS学习-类

TypeScript 中的类（Class）提供了面向对象编程的功能，包括继承、封装、多态等特性。与 JavaScript 的类相比，TypeScript 中的类更加强大，因为它具有类型检查、访问修饰符等特性，从而能够编写更健壮的代码。

类

类定义

任何实体都可以被抽象为一个使用类表达的类似对象的数据结构，这个数据结构既包含属性，又包含方法，在TypeScript 中使用 class 关键字定义类。

访问修饰符

访问修饰符用于控制类成员的可见性。TypeScript 中有三种修饰符：public、private 和 protected。

  1. public：默认修饰符，类的属性和方法可以在类的外部访问。
  2. private：私有属性或方法只能在类内部访问，不能在外部或子类中访问。
  3. protected：受保护的属性或方法只能在类内部和子类中访问，不能在类的外部直接访问。

类继承

TypeScript 支持类的继承。可以使用 extends 关键字来创建一个类的子类，子类中使用 super 关键字来调用父类的构造函数和方法。

抽象类

抽象类是一种只能被继承而不能直接实例化的类，用于定义类的模板。抽象类中的抽象方法没有具体实现，必须在子类中实现。

只读属性

TypeScript 支持 readonly 修饰符，用于定义只读属性。一旦初始化后，属性值就不能被修改。

存取器

TypeScript 支持 get 和 set 存取器，可以通过它们控制对属性的访问和修改。

静态属性和方法

使用 static 关键字可以定义静态属性和方法。静态成员属于类本身，而不是某个实例，可以直接通过类名访问。

类的类型（接口实现）

TypeScript 中的类可以实现接口，接口可以为类定义公共的结构和行为。

类的泛型

类可以使用泛型，从而可以在实例化时指定类型，使得类具有更强的灵活性。
