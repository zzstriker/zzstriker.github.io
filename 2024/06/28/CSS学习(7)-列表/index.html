CSS学习(7)-列表

列表的行为表现跟其他任何文本其实差不多，但需要了解一些专门用于列表的 CSS 样式

列表

无序、有序和描述列表，它们都具有相似的样式特性，而某些特性却又各不相同。

  1. <ul>和<ol>元素含有 16px 的顶部和底部的margin和 40px 的padding-left
  2. <li>元素默认是没有设置间距的
  3. <dl>元素含有 16px 的顶部和底部的margin，但不含内边距
  4. <dd>元素含有 40px 的margin-left

列表间距

通过 line-height 属性设置列表项之间的垂直间距

列表符号

符号样式

通过 list-style-type 属性设置列表符号的类型

属性值：

  1. none：不显示列表项的标记
  2. disc：实心圆点
  3. circle：空心圆点
  4. square：实心方块
  5. decimal：十进制阿拉伯数字
  6. lower-roman：小写罗马数字
  7. upper-roman：大写罗马数字

符号位置

list-style-position属性设置在每个列表项开始之前，列表符号是出现在列表项内，还是出现在其外

属性值：

  1. inside：使项目符号位于列表项内
  2. outside：使项目符号位于列表项之外

自定义符号

list-style-image属性允许对于列表符号使用自定义图片

列表样式简写

通过简写属性list-style来设置上面的三个属性，属性值可以任意顺序排列，你可以设置一个，两个或者所有三个值。如果指定了 type 和 image，如果由于某种原因导致图像无法加载，则 type 将用作回退。

管理计数

可以为<ol>有序列表设置计数规则

属性：

  1. start：一个整数值属性，指定了列表编号的起始值。此属性的值应为阿拉伯数字，不管list-style-type设置的是什么类型
  2. reversed：使列表反向计数

也可以为<ol>元素中的<li>元素指定数值

属性：

  1. value：设置列表项指定数值
