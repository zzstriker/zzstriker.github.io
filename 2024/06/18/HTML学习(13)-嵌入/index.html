HTML学习(13)-嵌入

图片、多媒体等内容丰富了HTML页面，如何在HTML页面展示其他类型的内容。

嵌入

使用嵌入元素可以在网页上嵌入各种类型的内容，<iframe>元素用于嵌入其他网页，<embed>和<object>元素用于嵌入外部资源。

iframe

<iframe>元素表示嵌套的浏览上下文，它能够将另一个HTML页面嵌入到当前页面中。

属性：

  1. src：被嵌套的页面的URL地址
  2. width和height：iframe的宽度和高度，默认是300*150
  3. allowfullscreen：设置为true时，通过调用**requestFullScreen()**方法激活全屏模式
  4. name：嵌入的浏览上下文的名称
  5. sandbox：控制应用于嵌入在 <iframe> 中的内容的限制（沙盒）

<embed>和<object>

<embed>和<object>元素用于嵌入外部资源。

<embed>

属性：

  1. src：被嵌套的资源的 URL。
  2. type：MIME 类型。
  3. width和height：资源显示的宽度和高度

<object>

属性：

  1. data：一个合法的 URL 作为资源的地址，需要为 data 和 type 中至少一个设置值。
  2. type：data 指定的资源的 MIME 类型，需要为 data 和 type 中至少一个设置值。
  3. width和height：资源显示的宽度和高度
