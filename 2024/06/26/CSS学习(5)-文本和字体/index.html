CSS学习(5)-文本和字体

元素中的文本是布置在元素的内容框中，通过使用 CSS 设置文本样式和字体。

文本和字体

用于样式文本的 CSS 属性通常可以分为两类

  1. 字体样式:：作用于字体的属性，会直接应用到文本中，比如使用哪种字体，字体的大小是怎样的，字体是粗体还是斜体等等。
  2. 文本布局风格: 作用于文本的间距以及其他布局功能的属性，比如，允许操纵行与字之间的空间，以及在内容框中，文本如何对齐。

字体

字体的属性可以通过 font 的简写方式来设置

font-family

使用font-family属性设置文本的字体

默认字体

  1. serif：衬线字体
  2. sans-serif：无衬线字体
  3. monospace：等宽字体，指包含的全部字符的宽度相同的字体
  4. cursive：手写字体
  5. fantasy：装饰字体

字体栈

由于无法保证在网页上使用的字体的可用性，可以提供一个字体栈，这样的话，浏览器就有多种字体可以选择了。只需包含一个 font-family 属性，其值由几个用逗号分离的字体名称组成。

font-size

使用 font-size属性来控制文本元素的大小

  1. px：将像素的值赋予给文本，这是一个绝对单位
  2. em：当前元素的父元素上设置的字体大小
  3. rem：HTML 中的根元素的字体大小

font-style

使用 font-style 属性设置文本是否应采用斜体。

  1. normal：将文本设置为普通字体
  2. italic：文本设置为斜体版本
  3. oblique：将普通文本倾斜的样式应用到文本中

font-weight

利用font-weight属性，设置文字的粗体大小。此属性接受关键字值（normal、bold）、相对关键字值（lighter、bolder）和数值（100 到 900）。

  1. normal和bold分别等同于数值 400 和 700。
  2. lighter和bolder将当前元素的粗体设置为比其父元素粗体更细或更粗一步。

文本

行高

line-height属性是指文本行基线之间的距离，指定line-height可以用来增加文本行之间的垂直间隔

水平对齐

text-align影响文本的水平对齐方式，只应用于块级元素，默认值为left。

  1. left：左对齐文本
  2. right：右对齐文本
  3. center：居中文字
  4. justify：两端对齐，使文本展开，改变单词之间的差距，使所有文本行的宽度相同。

垂直对齐

vertical-align属性只应用于行内元素，影响文本垂直方向对齐。

  1. baseline：基线对齐，默认元素的基线与其父元素的基线对齐
  2. sub：下标对齐，元素的基线与其父元素的下标基线对齐
  3. super：上标对齐，元素的基线与其父元素的上标基线对齐
  4. top：顶端对齐，将元素的顶部与整行的顶部对齐
  5. text-top：文字顶端对齐，将元素的顶部与父元素的字体顶部对齐
  6. middle：居中对齐，将元素的中点与父元素基线对齐再加上x-height/2
  7. bottom：底端对齐，将元素的底部与整行的底部对齐
  8. text-bottom：文字底端对齐，将元素的底部与父元素的字体底部对齐

字母和单词间距

单词间距

word-spacing 属性设置文本中的单词与单词之间的间距。

字母间距

letter-spacing 属性设置文本中的字母与字母之间的间距。

文本转换

使用 text-transform 属性来处理文本的大小写

  1. none：对文本不做任何改动
  2. uppercase：将文本转换为全部大写字符
  3. lowercase：将文本转换为全部小写字符
  4. capitalize：只对每个单词的首字母大写

文本装饰

使用 text-decoration 属性来处理文本的装饰

  1. none：取消已经存在的任何文本装饰
  2. underline：文本下划线
  3. overline：文本上划线
  4. line-through：穿过文本的线

text-decoration属性有更具体的详细写法：

  1. text-decoration-line：指定在文本何方增加装饰线
  2. text-decoration-color：用于设置text-decoration-line中所有装饰的颜色
  3. text-decoration-style：接受 solid、double、dotted、dashed和wavy关键字，用于描述装饰线的类型
  4. text-decoration-thickness：接受长度值，描述装饰线的宽度

文本缩进

使用 text-indent 向文本块添加缩进，属性采用长度或所在块宽度的百分比。

文本溢出

使用 text-overflow 指定隐藏内容的呈现方式。

  1. clip：在溢出点截断文本
  2. ellipsis：在溢出点显示省略号

空格换行

使用white-space属性处理元素中的空白和换行。

  1. normal：空格合并，换行会被当作空格来处理，允许自动换行
  2. nowrap：空格合并，换行会被当作空格来处理，不允许自动换行
  3. pre：空格保留，换行保留，不允许自动换行
  4. pre-wrap：空格保留，换行保留，允许自动换行
  5. pre-line：空格合并，换行保留，允许自动换行
  6. break-spaces：空格保留，换行保留，允许自动换行

单词拆分

使用word-break更改文本在溢出时如何换行

  1. normal：整个单词不拆分，直接换行
  2. break-all：将单词拆分，直到行尾再换行

文本方向

使用direction设置文本方向，ltr（从左到右）或 rtl（从右到左）
