CSS学习(9)-背景

可以用 CSS 背景做一些创造性事情，添加背景图片或者渐变，背景可以解决 CSS 中的很多样式问题。

背景

background 属性是背景属性的简写，用于为元素添加背景效果。

背景颜色

background-color 属性定义了 CSS 中任何元素的背景颜色。

背景图像

background-image 属性可以在一个元素的背景中显示一个图像。支持多背景图像设置，背景之间用逗号(,)隔开。

  1. 使用url()函数的图片地址
  2. 渐变函数

背景平铺

background-repeat 属性用于控制图像的平铺行为。

属性值：

  1. repeat：默认值，图像会按需重复来覆盖整个背景图片所在的区域。最后一个图像会被裁剪，如果它的大小不合适的话。
  2. repeat-x：仅水平方向上重复平铺
  3. repeat-y：仅垂直方向上重复平铺
  4. no-repeat：图像不会被重复
  5. space：图像会重复，但是不会裁剪。第一个和最后一个图像会被固定在元素相应的边上，同时空白会均匀地分布在图像之间。
  6. round：剩余的空间不够放置下一个图片的时候，重复的图片会伸展填充满空间。

背景定位

background-position 属性允许选择背景图片出现在它所应用的盒子上的位置。background-position 属性可以使用top、bottom、center、left 和 right 关键字，或者使用长度或百分比值。background-position 属性可以被指定一个或多个值。

  1. 一个值：
    • center：居中背景图片
    • top、bottom、left、right：指定背景图片放在哪个边，另一个维度被设置为50%
    • 长度、百分比：相对于左边的x坐标，y坐标被设置为50%
  2. 两个值：
    • top、bottom、left、right、center：一个定义x坐标，另一个定义y坐标，同垂直或水平方向的关键字使用两个是无效的，例如：top top 或 left right
    • 长度、百分比：如果第一个值是关键字，那么定义该关键字另一方向的位置，如果两个值都是长度和百分比，则第一个定义x坐标，第二个定义y坐标
  3. 三个值：
    • 第一个值是关键字 top、left、bottom、right，或者 center。
    • 如果第二个值是长度或百分比，则是第一个值的偏移量。如果是第三个值，则是第二个值的偏移量。
  4. 四个值：第一个和第三个值是定义x和y的关键字值。第二个和第四个值是前面x和y关键字值的偏移量

背景大小

background-size 属性设置背景图片大小。图片可以保有其原有的尺寸，或者拉伸到新的尺寸，或者在保持其原有比例的同时缩放到元素的可用空间的尺寸。设置背景图片的大小只能通过关键字或者设置宽高来实现。通过宽度和高度值来设定尺寸时，可以提供一个或两个数值，如果仅有一个数值，该数值将作为宽度大小，高度被设定为 auto，如果有两个数值，第一个作为宽度大小，第二个作为高度大小。

属性值：

  1. auto：以背景图片的比例缩放背景图片。
  2. contain：缩放背景图片以完全装入背景区，可能背景区部分空白。
  3. cover：缩放背景图片以完全覆盖背景区，可能背景图片部分看不见。
  4. 长度：指定背景图片大小，不能为负值。
  5. 百分比：指定背景图片相对盒子的百分比。

渐变

渐变是由gradient()函数创建的，渐变是一种图片，由至少两种或多种颜色之间的渐变过渡构成。

线性渐变

线性渐变创建了一条沿直线前进的颜色带。

基础渐变

最基本的渐变类型，只需指定两种颜色即可。

渐变方向

默认情况下，线性渐变的方向是从上到下，可以指定一个值来改变渐变的方向。

渐变角度

如果想要更精确地控制渐变的方向，可以给渐变设置一个具体的角度。在使用角度的时候，0deg 代表渐变方向为从下到上，90deg 代表渐变方向为从左到右，诸如此类正角度都属于顺时针方向。而负角度意味着逆时针方向。

自定义渐变

默认情况下，无论设置多少颜色，所设置颜色会均匀分布在渐变路径中。

渐变会平滑地从一种颜色过渡到另一种颜色，可以通过设置一个值在颜色之间达到控制渐变进程的效果。

渐变还支持透明度，可用于创建淡入淡出效果。

叠加渐变

可以将多个背景叠加起来以实现一些非常好看的效果。背景是从顶部到底部堆叠的，指定的第一个就是顶部。

径向渐变

径向渐变是从一个中心点向外辐射的，可以指定中心点的位置，还可以使其为圆形或者是椭圆形。

基础渐变

最基本的渐变类型，只需指定两种颜色即可。默认情况下，渐变的中心点是 50% 50% 的位置，径向渐变是匹配盒子的比例显示是圆形还是椭圆形。

渐变位置

可以用百分比或者绝对长度指定每个径向渐变颜色的位置。

渐变中心

可以使用关键字、百分比或者绝对长度、长度以指定渐变的中心。

渐变大小

径向渐变可以指定渐变的大小，通过 closest-corner、closest-side、farthest-corner 和 farthest-side，其中 farthest-corner 是默认值。圆的大小为长度，椭圆则是长度和百分比。

  1. closest-side：渐变结束形状如果是圆形，与容器距离渐变中心点最近的一边相切，如果是椭圆，则与距离渐变中心点最近的垂直或水平边相切。
  2. closest-corner：渐变结束形状与容器距离渐变中心点最近的一个角相交。
  3. farthest-side：渐变结束形状与容器距离渐变中心点最远的一个角相交。
  4. farthest-corner：默认值，渐变的结束形状与容器距离渐变中心点最远的一个角相交。
