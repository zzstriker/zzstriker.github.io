HTML学习(9)-表单

表单是用户和服务器之间交互的主要内容之一，表单允许用户输入数据，大多数情况下会将数据发送到web服务器进行处理和存储。

表单

表单是由一个或多个表单控件组成，这些控件可以是文本字段、选择框、按钮、复选框或单选按钮等等。

<form>元素

创建一个HTML表单都必须从<form>元素开始，<form>元素包含各个表单控件，用于向 Web 服务器提交信息。

属性：

  1. action：表单提交的 URL
  2. method：浏览器使用哪种HTTP方法来提交表单

<fieldset>和<legend>元素

出于样式和语义的目的，<fieldset>元素用于对表单中的控件进行分组。

<legend>元素用于表示父元素<fieldset>内容的标题。

<label>元素

<label>元素用于表示表单中某个控件的说明。

属性

  1. for：用于关联表单控件id属性的值，或者使用<label>元素直接包裹表单元素，也可以达到for属性的作用。

<input>元素

<input>元素可以让用户输入任何类型的数据，在<input>元素中，最重要的属性是type属性，type属性定义了<input>元素各种使用方式。

属性 类型 描述
type all 表单控件的类型
name all 表单控件的名称
value all 表单控件的初始值
disabled all 表单控件是否禁用
autocomplete 除checkbox、radio、button、file、hidden以外均有效 自动补全
checked checkbox、radio 控件是否选中
accept file 可选择文件类型的列表
max time、number、range 最大值
min time、number、range 最小值
maxlength text、search、url、tel、email、password value的最大长度（字符数）
minlength text、search、url、tel、email、password value的最小长度（字符数）
multiple email、file 是否多选
pattern text、search、url、tel、email、password 为了使得 value 有效，必须符合的模式
placeholder text、search、url、tel、email、password、number 当没有值设定时，出现在表单控件上的文字
readonly 除checkbox、radio、button、hidden、range、color以外均有效 如果存在，其中的值将不可编辑。
required 除button、hidden、range、color以外均有效 如果存在，一个值是必需的，或者必须勾选该值才能提交表格。
size text、search、url、tel、email、password 控件的尺寸
step time、number、range 有效的增量值

<button>元素

<button>元素表示一个可点击的按钮。

属性：

  1. type：button 的类型
    • submit：按钮提交表单数据给服务器
    • reset：按钮重置所有表单组件至初始值
  2. formaction：按钮提交表单信息的URI，如果使用了该属性，将覆盖<form>元素的action属性
  3. formmethod：按钮提交表单使用的HTTP方法，，如果使用了该属性，将覆盖<form>元素的method属性

文本域

<textarea>元素表示文本域，文本域与单行文本框的区别在于，用户可以在要提交的数据中使用换行。

属性：

  1. rows：文本域指定的可见文本的行数，值为正整数，默认为2
  2. cols：文本域的可见宽度，以平均字符宽度为单位，值为正整数，默认为20

下拉框

下拉菜单是一种简单地让用户从许多不同选项中进行选择的控件，不占用用户界面太多空间。HTML 有两种下拉内容：一种是选择框、另外一种是自动补全框。

选择框

选择框是由<select>元素及一个或多个<option>子元素构成的，每个子元素指定了选择框的可能取值。

属性：

  1. multiple：在<select>元素设置multiple属性，选择框就允许用户选择多个选项
  2. size：控件中同时可见的行数
  3. value：在<option>元素设置value属性，当表单提交时也会提交这个选项对应的值，如果省略了value属性，<option>元素的内容会作为提交的值。

<option>元素可以嵌套在<optgroup>元素中，从视觉上给选项做分组。在<optgroup>元素中，label属性的值在嵌套选项前显示。

自动补全框

通过<datalist>元素以及<option>子元素为表单项提供推荐的自动补全值。<datalist> 元素需要指定一个id。<datalist>元素的id会绑定<input>元素的list属性，绑定之后<datalist>元素的选项就会被用来自动补全用户输入的文本。
