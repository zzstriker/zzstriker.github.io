CSS学习(2)-选择器

CSS 选择器用来指定网页上我们想要样式化的 HTML 元素，通过 CSS 选择器可以很精细的处理需要样式化的元素。

选择器

CSS 选择器是 CSS 规则的第一部分。它是元素和其他部分组合起来告诉浏览器哪个 HTML 元素应当是被选为应用规则中的 CSS 属性值的方式。选择器所选择的元素，叫做“选择器的对象”。

通配选择器

显示为一个星号，这个选择器可以与任何元素匹配，就像是一个通配符。

元素选择器

元素选择器也叫做标签选择器或者是类型选择器，HTML 元素的名字就是元素选择器的名字

类选择器

类选择器以一个句点（.）开头，会选择文档中应用了这个类的所有元素。

特点：

  1. 一个元素可以应用多个类选择器
  2. 一个类选择器也可以在多个元素中应用

ID选择器

ID选择器以#开头，在一个HTML文档中，ID选择器的id有且仅能使用一次

特点：

  1. 一个元素只有一个id属性
  2. id属性的值在一个 HTML 文档中唯一

属性选择器

属性选择器查找具有特定 HTML 属性或具有特定 HTML 属性值的元素。 用方括号 ([ ]) 将选择器括起来，指示 CSS 查找属性。

选择器 描述
[attribute] 匹配带有一个名为attribute的属性的元素——方括号里的值。
[attribute = value] 匹配带有一个名为attribute的属性的元素，其值为value——引号中的字符串。
[attribute ~= value] 匹配带有一个名为attribute的属性的元素，其值有一个或者更多，至少有一个和value匹配。
[attribute ^= value] 匹配带有一个名为attribute属性的元素，其值开头为value子字符串。
[attribute $= value] 匹配带有一个名为attribute属性的元素，其值结尾为value子字符串
[attribute *= value] 匹配带有一个名为attribute属性的元素，其值的字符串中的任何地方，至少出现了一次value子字符串。

伪类

伪类是选择器的一种，伪类选择器始终以一个冒号(:)开始。它用于选择处于特定状态的元素。

用户行为伪类

  1. :hover：设置元素在其鼠标悬停时的 CSS 样式
  2. :focus：设置元素在其拥有输入焦点时的 CSS 样式
  3. :link：设置 a 元素在未被访问前的 CSS 样式
  4. :visited：设置 a 元素在其链接地址已被访问过时的 CSS 样式
  5. :active：设置元素在被用户激活时的 CSS 样式

文档结构伪类

  1. :first-child：第一个子元素
  2. :last-child：最后一个子元素
  3. :nth-child(n)：正着数第几个子元素，n可以是数字，也可以是关键字（odd 奇数，even 偶数），或者是公式
  4. :nth-last-child(n)：倒着数第几个子元素，n可以是数字，也可以是关键字（odd 奇数，even 偶数），或者是公式

伪元素

伪元素表现得是像往标记文本中加入全新的 HTML 元素一样，而不是向现有的元素上应用类。伪元素开头为双冒号（::）。

  1. ::first-letter：元素的第一个字母。
  2. ::first-line：包含此伪元素的元素的第一行。
  3. ::before：出现在原有元素的实际内容之前
  4. ::after：出现在原有元素的实际内容之后

关系选择器

后代选择器

用单个空格（" "）字符组合两个选择器，匹配的目标元素是其他元素的后代，只要是后代即可

子代选择器

子代关系选择器是个大于号（>），只会在选择器选中直接子元素的时候匹配。继承关系上更远的后代则不会匹配。

相邻兄弟选择器

相邻兄弟选择器是个加号（+），匹配的目标元素紧跟在其他元素后面。

通用兄弟选择器

如果想选中一个元素的兄弟元素，即使它们不直接相邻，可以使用通用兄弟关系选择器（~）。
