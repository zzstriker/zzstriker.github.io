CSS学习(1)-CSS介绍

层叠样式表—–Cascading Style Sheets，CSS用来指定文档如何展示给用户的一门语言——如网页的样式、布局、等等。

CSS语法

CSS 是一门基于规则的语言，你能定义用于你的网页中特定元素样式的一组规则。CSS由2个部分组成：

  1. 属性：可读的标识符，指示想要更改的样式特征。
  2. 值：每个指定的属性都有一个值，这个值表示如何对属性施加样式。

当一个属性与一个值配对时，这种配对被称为 CSS 声明。

CSS 声明可以在 CSS 声明块中找到。

CSS 声明块与选择器配对，生成 CSS 规则集（或称 CSS 规则）。

CSS引用方式

外部样式表

外部样式表在一个单独的扩展名为 .css 的文件中包含 CSS。这是将 CSS 应用到文档中最常见和最有用的方法。你可以将一个 CSS 文件链接到多个网页上，用同一个 CSS 样式表为所有网页确定样式。使用 HTML <link> 元素来链接外部样式表文件

内部样式表

内部样式表在 HTML 文档内。创建一个内部样式表，要把 CSS 放置在包含在 HTML <head> 元素中的 <style> 元素内。

内联样式

内联样式是影响单个 HTML 元素的 CSS 声明，包含在元素的 style 属性中。

注释

CSS 中的注释以 /* 开头，以 */ 结尾。
