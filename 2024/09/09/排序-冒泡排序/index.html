排序-冒泡排序

冒泡排序（Bubble Sort）是一种基础的排序算法。它通过多次遍历数组，依次比较相邻的两个元素，如果它们的顺序错误（如：从大到小排序时，前一个元素大于后一个元素），则交换它们的位置。每一轮遍历结束后，最大的元素会逐渐 “冒泡” 到数组的末尾。这个过程会重复多次，直到数组完全有序。

冒泡排序

冒泡排序步骤

  1. 从数组的第一个元素开始，比较相邻的两个元素。
  2. 如果前一个元素大于后一个元素，则交换它们的位置。
  3. 对每一对相邻元素重复这个操作，直到到达数组的末尾。
  4. 每一轮遍历之后，最大的元素被放到数组的最后一个位置。
  5. 重复这个过程，忽略已经排好序的部分，直到整个数组有序。

代码实现

  1. 外层循环 (for (let i = 0; i < len - 1; i++))：控制整个排序过程的轮数。每一轮都会把最大的元素放在末尾。
  2. 内层循环 (for (let j = 0; j < len - 1 - i; j++))：在每一轮中，逐个比较相邻元素，交换顺序不对的元素。注意每一轮中，后面已经排好序的部分不再需要比较，因此 j 的范围逐渐减小。
  3. 交换元素：[arr[j], arr[j + 1]] = [arr[j + 1], arr[j]] 是通过数组解构的方式交换元素的位置。

代码优化

冒泡排序在最坏情况下（即数组完全逆序的情况下）需要进行多次遍历。可以通过添加一个标志变量 swapped 来优化，若某一轮遍历中没有发生元素交换，说明数组已经有序，可以提前结束排序。

时间复杂度

  1. 最坏情况时间复杂度：O(n^2)，数组完全逆序时，需比较每一对相邻元素并进行交换。
  2. 最优情况时间复杂度：O(n)，如果数组已经有序，只需要进行一次遍历即可。
  3. 空间复杂度：O(1)，只需要常数级的额外空间用于临时变量交换元素。
